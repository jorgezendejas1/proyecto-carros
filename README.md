# Proyecto de Anuncios de Coches - Sprint 7

Este proyecto consiste en una aplicación web interactiva que analiza un conjunto de datos de anuncios de venta de coches en los EE. UU.

## Funcionalidades
- Visualización de un histograma sobre el kilometraje (odómetro) de los vehículos.
- Visualización de un gráfico de dispersión que relaciona el precio y el kilometraje.
- Interfaz interactiva construida con Streamlit y desplegada en la nube.

## Herramientas utilizadas
- Python, Pandas, Plotly, Streamlit.
- Entornos virtuales (venv).
- Despliegue en Render.
